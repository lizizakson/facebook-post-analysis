# Facebook Post Performance Analysis

Exploratory analysis of Facebook post performance using data exported from the Meta dashboard.

## Goals

Understand what influences post performance:

- posting time
- post type
- audience engagement
- algorithmic exposure

## Methods

The analysis includes:

- data cleaning
- engagement rate calculation
- exploratory data analysis
- visualization of engagement vs exposure
- comparison across posting times and post types

## Tools

- Python
- Pandas
- Seaborn
- Jupyter Notebook

## Structure

data/ – exported Facebook data  
code/ – analysis scripts  
facebook_post_analysis.ipynb – main notebook
